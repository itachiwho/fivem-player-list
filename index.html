Notices & Features

Duplicate Names

- Players with the same FiveM name (e.g., ID 223 “USER” and ID 376 “USER”) and also in the sheet list will both appear in Shift Group as duplicate if the name matches.

Shift Groups Source

- Shift groups load from Google Sheets . If that sheet is slow or changes format, roles may briefly show as “-” until it loads.

Refresh & Updates

- The list auto-refreshes every 30s. You can also click the refresh icon. “Last updated” time shows the most recent successful fetch.

Search & Filters

- Search matches by player Name and ID. The Shift filter matches exact names in the sheet (case sensitivity depends on sheet entries).
